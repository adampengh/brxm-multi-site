# Running Project Locally

## Backend (brXM)
```
cd backend
mvn clean verify
mvn -P cargo.run
```

## Frontend (React)
React CSR App runs at http://localhost:3000
```
cd frontend/react
yarn
yarn start
```

## GraphQL
Apollo GraphQL Service runs at http://localhost:4000/graphql
```
cd brx-graphql
yarn
yarn start
```
